Jquery validator

Plugin validación de formularios

El proyecto contiene un plugin para realizar validaciones de captura de información desde el cliente, la librería se llama Jquery Validator (ubicada en "/recursos/assets/jquery-validator/*")

Ya existe una librería que integra validaciones personalizadas e inicializa el control a etiquetas en espa�ol, por default ya se inicializa en el coreLayout.html, el nombre del archivo se llama "moduloValidador.js". (Ubicada en "/recursos/script/comun/moduloValidador.js).

Las validaciones personalizadas adicionales a las que ofrece el plugin.

  • soloNumeros
  • soloDecimales
  • soloCaracteres
  • formatoFechas
  • noVacio
  • validaExtensionJpg
  • fechaActualMaximo

  • Consultar plugin Jquery Validator

///Inicializacion del formulario a validar
function ..... (){
	$("#formulario-validado").validate({
		rules: {
			"email": {
				"email": true
			},"select":{
				"required":true
			},"contrasenia":{
				"required":true,
				"minlength" : 3                	
			},"nombre":{
				"soloCaracteres":true          	
			},"numeros":{
				"soloNumeros": true
			},"archivo":{
				"required": true
			},"texto" : {
				"maxlength" : 20 
			}
		},
		messages: {
			"archivo" : {
				"required" : "Debe adjuntar un archivo..."
			}
		}
	});
};
	
	
///Ejecución de evento para validar formulario previamente configurado
function ... (){
	$("#btn-validar").click(function(){
		var capturaValida = $("#formulario-validado").valid();
		if(capturaValida){
			alert("La captura es valida");
		}		
	});		
};
